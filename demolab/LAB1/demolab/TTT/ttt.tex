\section*{Tic-Tac-Toe}

Tic-tac-toe går ut på att få tre egna symboler i rad. Du kan
välja att spela mot ett tidigare tränat, ett av dig tränat eller ett
otränat system. 
  
När du tränar systemet lär sig systemet spelställningar, vilka drag som
gjorts där och vem som vann (belöning). Systemet försöker sedan i
varje drag maximera sin belöning på lång sikt. Denna typ av inlärning
brukar kallas betingad inlärning (reinforcement learning) och
kännetecknas av problem där man inte vet hur man ska gå för att komma
fram till målet (ex att vinna). Detta till
skillnad från lärarleddinlärning där man vet vad varje insignal ska ge
för utsignal. 

Starta programmet genom att skriva 'tictactoe' på kommandoraden i matlab.
